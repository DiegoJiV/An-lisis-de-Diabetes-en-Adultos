Análisis de Diabetes en Adultos

El proyecto se basa en datos de la Encuesta de Salud y Nutrición, utilizando la información para desarrollar modelos descriptivos para analizar los datos y
así hacer predicciones para pacientes futuros en base a datos del adulto.

En la carpeta Datos (Originales) se encuentran los datos sin procesar. Estos son los que se usan en el archivo E2_EDA_preprocesamiento.ipynb.
En la carpeta Datos (Procesados) se encuentran los datos despues del procesamiento del E2 y son los que se usan para el archivo E3_modelacion_evaluacion.ipynb. 

En el archivo de cuaderno jupyter E2_EDA_preprocesamiento.ipynb se visualiza el preprocesamiento y analisis de los datos.

En el archivo E3_modelado_evaluacion se desarrollan los modelos para la predicción y análisis de diabetes en adultos, utilizando los datos preprocesados en
E2_EDA_preprocesamiento.ipynb.

Propuesta_Proyecto.pdf es el entregable 1, donde se muestran el problema, descripcion de los datos, y el rol de cada integrante.

E4_Informe_final.pdf muestra un descripción del proyecto y los resultados obtenidos a traves de las diferentes etapas.
