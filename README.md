Análisis de Diabetes en Adultos
El proyecto se basa en datos de la Encuesta de Salud y Nutrición, utilizando la información para desarrollar modelos descriptivos para analizar los datos y
así hacer predicciones para pacientes futuros en base a datos del adulto.
El archivo Propuesta_Proyecto introduce la problemática del proyecto y los diferentes métodos que se utilizarán.
En los archivos mineria_principal.csv y mineria_enriquecido.csv se juntan todos los datos de las encuestas de ENSANUT 
(actividad_fisica_ensanut2024_w-adultos.csv, antropometria_ensanut2024_w.csv, ensasangre24.determinaciones_micronutrimentos.csv,
ensasangre24.determinaciones_micronutrimentos.csv).

En el archivo de cuaderno jupyter E2_EDA_preprocesamiento.ipynb se visualiza el preprocesamiento de los datos para ser posteriormente analizados.
En el archivo E3_modelado_evaluacion se desarrollan los modelos para la predicción y análisis de diabetes en adultos, utilizando los datos preprocesados en
E2_EDA_preprocesamiento.ipynb.
